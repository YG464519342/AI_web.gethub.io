# 🚀 快速开始指南

## 欢迎使用趣味答题平台!

这是一个专为儿童和青少年设计的趣味学习平台,采用可爱的3D黏土风格。

## 📖 使用步骤

### 1. 打开首页
在浏览器中访问 `http://localhost:8000` 或直接打开 `index.html`

### 2. 选择科目
点击任意科目卡片(如"数学")开始答题

### 3. 开始答题
- 阅读题目
- 点击选择答案(或按键盘1-4)
- 点击"提交答案"(或按Enter键)
- 查看反馈和解析
- 点击"下一题"继续(或按Space键)

### 4. 查看结果
完成所有题目后,查看你的得分、正确率和用时

### 5. 再次挑战
点击"再来一次"重新答题,或返回首页选择其他科目

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| 1-4 | 选择选项A-D |
| Enter | 提交答案 |
| Space | 下一题 |
| Tab | 切换焦点 |

## 🎯 答题技巧

1. **仔细阅读题目** - 不要着急,理解题意很重要
2. **注意时间** - 每题30秒,合理分配时间
3. **看清选项** - 有些选项很相似,要仔细区分
4. **学习解析** - 答错了不要灰心,看看解析学习知识点
5. **多次练习** - 题目顺序每次都不同,可以反复练习

## 📊 计分规则

- 每题10分
- 答对得分,答错不扣分
- 满分100分
- 根据得分获得不同评价:
  - 100分: 🏆 完美!全部答对!
  - 80-99分: 🎉 太棒了!
  - 60-79分: 👍 不错哦!
  - 0-59分: 💪 继续努力!

## 🎨 界面说明

### 首页
- **Hero区域**: 平台介绍和主按钮
- **科目卡片**: 6个科目可选(目前数学可答题)
- **进度演示**: 展示学习旅程
- **排行榜**: 查看高分榜
- **报名CTA**: 注册入口

### 答题页
- **顶部导航**: 显示科目和当前得分
- **进度条**: 显示答题进度
- **题目卡片**: 显示题目和选项
- **计时器**: 30秒倒计时
- **反馈区**: 显示答对/答错提示
- **操作按钮**: 提交/下一题/完成

## 🔧 自定义题库

想添加自己的题目?编辑 `assets/js/quiz.js` 文件:

```javascript
const questionBank = {
  math: [
    {
      question: "你的题目?",
      options: ["选项A", "选项B", "选项C", "选项D"],
      correct: 0,  // 正确答案索引(0-3)
      explanation: "答案解析"
    },
    // 添加更多题目...
  ]
};
```

## 🐛 常见问题

**Q: 为什么其他科目点击后还是数学题?**
A: 目前只实现了数学题库,其他科目的题库需要自己添加。

**Q: 可以修改每题的时间限制吗?**
A: 可以!在 `quiz.js` 中修改 `gameState.timeLeft = 30` 这个值。

**Q: 如何添加更多题目?**
A: 在 `quiz.js` 的 `questionBank` 对象中添加题目数据。

**Q: 可以关闭计时器吗?**
A: 可以注释掉 `startTimer()` 函数的调用,或将时间设置为很大的值。
